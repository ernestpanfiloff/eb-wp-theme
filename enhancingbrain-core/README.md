# Enhancing Brain Core (Plugin)

Companion plugin for the Enhancing Brain theme.

## What it adds

- Gutenberg block: `EB Disclaimer`
- Shortcode: `[eb_disclaimer]`
- Shortcode: `[eb_callout title="Note"]Your content[/eb_callout]`

## Install

1. Copy the `enhancingbrain-core` folder to `wp-content/plugins/`.
2. Activate **Enhancing Brain Core** in WordPress admin.

## Usage

- In the block editor, insert **EB Disclaimer**.
- In content, use:
  - `[eb_disclaimer]`
  - `[eb_disclaimer text="Custom disclaimer text."]`
  - `[eb_callout title="Research Note"]Always review primary sources.[/eb_callout]`
